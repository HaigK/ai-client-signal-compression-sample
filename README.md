# AI Client Signal Compression Sample

This is a Python/JSON/Markdown work sample that converts messy unstructured business text into structured client signals.

The sample uses fake client call notes, email threads, and support notes. It extracts pain points, risk signals, interest areas, recommended next actions, and simple scoring outputs.

## Purpose

This project demonstrates:

- AI workflow design
- Context engineering
- Unstructured text processing
- Knowledge compression
- JSON output design
- Business signal extraction
- Role-based workflow thinking

## Use Case

A sales, research, or client-success team may have long call transcripts, emails, and support notes. Important client signals can be buried in that text.

This sample shows how to compress that text into structured decision support.

## Technologies

- Python
- JSON
- Markdown
- Basic NLP-style keyword extraction
- Structured output formatting
